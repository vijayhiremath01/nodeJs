ES6 modules are used when prefer import and export these are the part of modern javascript we can use it in nodeJs by using or setting in package.json as "type" : "module"
export: Used to export functions or variables.
import: Used to import the exported functions/variables.

Reusability: Once written, the code can be reused across multiple files.
Maintainability: Breaking the code into modules makes it easier to manage and debug.
Encapsulation: Modules maintain their own independent scope.
Modular Design: Dividing the application into smaller parts makes the development process more streamlined
